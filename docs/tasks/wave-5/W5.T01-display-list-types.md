# Implement Display List Types

## Context

Display list is a flat list of draw commands (rectangles, text, images) generated from fragment tree. Optimized for painting.

## Implementation

```rust
pub enum DisplayItem {
    Rectangle { rect: Rect, color: Color, border_radius: f32 },
    Text { position: Point, glyphs: Vec<GlyphInstance>, color: Color },
    Image { rect: Rect, image_data: Arc<ImageData> },
    BoxShadow { rect: Rect, blur: f32, spread: f32, color: Color },
}

pub struct DisplayList {
    pub items: Vec<DisplayItem>,
}
```
