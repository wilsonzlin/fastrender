shape-outside: circle()

This fixture exercises CSS Shapes text wrapping. The floated square remains rectangular, but the line boxes should wrap against a circular contour with a small shape-margin. The tell-tale signal is that the first few lines start closer to the float than a pure rectangle would allow, then gradually widen as they flow past the circle.

FastRender previously treated floats as strictly rectangular, so the left edge of each line would jump to a constant offset and stay there for the full height of the float. With shape-outside wired into float layout, the available inline width should vary by scanline and more closely match Chrome.
